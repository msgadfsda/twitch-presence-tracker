Twitch Presence Tracker

Tracks inferred Twitch audience presence (join/leave), with popular visitor ranking, follower counts, and session durations.

Note: GitHub Pages hosts static files only. The tracker backend/API must run locally or on a server.

Run locally

npm install
npm run dev

Then open http://localhost:8787.

OAuth setup

Set environment variables:

TWITCH_CLIENT_ID=...
TWITCH_CLIENT_SECRET=...
TWITCH_REDIRECT_URI=http://localhost:8787/auth/callback
